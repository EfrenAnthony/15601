Práctica, ¿Que modelo de automóvil tiene un descuento?

El usuario ingresará la marca y el modelo de un auto, si la marca y el modelo es Ford Fiesta , se le muestre un mensaje de que se le otorga un descuento del 5%, Si la marca y el modelo es Ford Focus, se le de un descuento del 10 %, para cualquier otro modelo o marca se le asigne un descuento del 3%,
